## Creating an Order

To create a new order, you can use our API endpoint:

## Retrieving Orders

To retrieve a list of orders, use the following endpoint:

<Endpoint method="GET" url="/v1/orders" />

## Order Status

Orders can have the following statuses:

- Pending
- Processing
- Shipped
- Delivered
- Cancelled

## Order Details

Each order contains the following information:

- Order ID
- Customer Information
- Items Ordered
- Total Amount
- Order Date
- Shipping Address
- Payment Method
