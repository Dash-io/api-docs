title: "Report Overview"
api: "GET /reports"
---

### 📝 Summary

This endpoint allows users to update an existing expense record. Users can modify various attributes of an expense, such as the amount, expense type, date, department, and additional notes. Upon successful update, the system returns the updated details of the expense.


---

### 📥 Sample Request
#### Body Parameters
<Expandable>
**Description:** Success response containing a message, user data (including UUID), and status.

| **Field**   | **Type** | **Required** | **Description**          |

| `data.uuid` | string   | ✅ Yes       | Unique identifier for user |
