---
title: "Delivery Interests"
api: "GET /metas/delivery_interests"
---

### 📝 Summary

This endpoint retrieves a list of predefined delivery interests or preferences configured in the system. These interests represent specific categories or priorities for deliveries, such as express shipping, fragile items, or temperature-sensitive goods.


---

### 📥 Sample Request
#### Body Parameters
<Expandable>
**Description:** Success response containing a message, user data (including UUID), and status.

| **Field**   | **Type** | **Required** | **Description**          |

| `data`      | object   | ✅ Yes       | User data                 |
| `data.uuid` | string   | ✅ Yes       | Unique identifier for user |
| `status`    | string   | ✅ Yes       | Status of the request     |
</Expandable>
