---
title: "Delete Expense"
api: "Delete /expenses/:uuid"
---

### 📝 Summary

This endpoint allows users to delete an existing expense record from the system. Once the expense is deleted, it will no longer appear in the records, and the system will confirm the deletion.


---

#### Body Parameters
<Expandable>
