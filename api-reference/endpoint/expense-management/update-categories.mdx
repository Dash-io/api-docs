---
title: "Update Category"
api: "PATCH /expenses/categories/:uuid"
---

### 📝 Summary

This endpoint allows users to update an existing expense category in the system. Users can modify attributes such as the category name, description, or its active status. After a successful update, the system will return the updated details of the category.

---

##### Body Parameters
<Expandable>
