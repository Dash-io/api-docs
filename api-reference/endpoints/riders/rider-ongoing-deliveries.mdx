---
title: "Rider Ongoing Deliveries"
api: "GET /riders/ongoing-deliveries{{uuid}}"
---

### üìù Summary

This endpoint retrieves a list of ongoing deliveries assigned to a specific rider. Ongoing deliveries are those that are currently in progress or pending completion. This information helps track real-time deliveries for active riders.

---

### Requests
##### Params

<Expandable>
**Description:** 

|Name|Location|Type|Required|Description|
|---|---|---|---|---|
|`uuid`|path|string| yes |none|
</Expandable>

##### Example

<Expandable>
```json
{
  "uuid": "6b8b5c9f-9182-407d-85cb-7fce108620e9"
}
```
</Expandable>
